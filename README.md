# Gothic Jewelry Shop

A web application — an online store for gothic jewelry, developed using the Django framework.  
The project is designed to showcase and sell gothic-style jewelry with the ability to manage products through an administrative panel.

---

## Project Description

Gothic Jewelry Shop is a website that allows users to:
- browse a catalog of gothic jewelry
- view detailed information about each product
- register and log in
- place orders
- manage products and orders through the admin panel

The project is built using the MTV (Model–Template–View) architecture.
